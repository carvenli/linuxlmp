LinuxLMP 使用教學

安裝eAccelerator : sh /root/linuxlmp/eaccelerator.sh

安裝Zend Optimizer : sh /root/linuxlmp/zend_optimizer.sh

安裝ionCube Loader : sh /root/linuxlmp/ioncube.sh

安裝Google Performace Tools : sh /root/linuxlmp/perftools.sh

安裝FTP(vsftpd) : sh /root/linuxlmp/vsftpd.sh

添加FTP帳號 : sh /root/linuxlmp/vsftpd.sh

安裝Nginx前端 : sh /root/linuxlmp/nginx.sh

創建Litespeed虛擬主機 : sh /root/linuxlmp/vhost.sh

*創建Nginx虛擬主機 : sh /root/linuxlmp/nginx.sh

進行備份 : sh /root/linuxlmp/backup.sh

重啟Litespeed : /etc/init.d/lsws restart

重啟MySQL : /etc/init.d/mysql(d) restart

重啟Nginx : /etc/init.d/nginx restart

Litespeed後台 : http://{ip}:7080

phpMyAdmin : http://{ip}/phpmyadmin/

php.ini目錄 : /usr/local/lsws/lsphp5/lib/php.ini

MySQL my.cnf : /etc/my.cnf (Centos) | /etc/mysql/my.cnf (Debian/Ubuntu)

Nginx conf檔案 : /etc/nginx/nginx.conf 及 /etc/nginx/conf.d/下

虛擬主機目錄: /home/wwwroot

虛擬主機文件權限修正 : chown -R nobody:nobody /home/wwwroot/域名/html

LLsMP安裝記錄 : /root/linuxlmp/install.log

*若安裝了Nginx前端，需先創建Litespeed 虛擬主機，再創建Nginx虛擬主機
